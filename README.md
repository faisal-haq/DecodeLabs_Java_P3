 # 🏧 ATM Interface (Core Java)

A simple console-based ATM Interface application developed using Core Java. This project simulates basic ATM operations such as depositing money, withdrawing money, checking account balance, and exiting the system. It also includes input validation using exception handling for a smooth user experience.

---

## 🚀 Features

- 💳 ATM card insertion simulation
- 💰 Deposit money
- 💸 Withdraw money
- 💵 Check account balance
- ⚠️ Input validation using `InputMismatchException`
- ✅ Handles invalid menu choices
- 🖥️ Console-based interactive interface
- 🔒 Uses encapsulation with a private balance variable
- 📋 Menu-driven program using loops and switch-case

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/ATM-Interface.git
```

2. Open the project in VS Code or any Java IDE.

3. Compile the program

```bash
javac Decodelabs_Java_p3.java
```

4. Run the program

```bash
java Decodelabs_Java_p3
```
